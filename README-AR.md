🎯 الرؤية

DRS.VIP-AI ليس مجرد واجهة محادثة، بل هو بيئة تشغيل أوامر للذكاء الاصطناعي متكاملة مصممة لتكون أكثر واجهة ذكاء اصطناعي تقدمًا في العالم حتى عام 2035.

يجمع النظام بين قوة:

ChatGPT — ذكاء محادثة متقدم

JARVIS — قدرات مساعد ذكي

مركز قيادة للأمن السيبراني — مراقبة أمنية بمستوى دفاعي

محرك تصور عصبي — رسومات عصبية ثلاثية الأبعاد في الوقت الحقيقي

وحدة تطوير ذكاء اصطناعي — بيئة تطوير كاملة

واجهة تحكم WebGL متقدمة — تصورات مرئية مع تسريع عبر GPU

🛠️ التثبيت

المتطلبات

Python 3.11 أو أحدث

Node.js 20+ (لأدوات التطوير الاختيارية)

Ollama لتشغيل نماذج الذكاء الاصطناعي محليًا

متصفح حديث يدعم WebGL2



---

التشغيل السريع

# استنساخ المستودع
git clone https://github.com/drs-vip-1/drs-vip-ai.git
cd drs-vip-ai

# إنشاء بيئة افتراضية
python -m venv venv
source venv/bin/activate

# تثبيت المتطلبات
pip install -r requirements.txt

# إعداد البيئة
cp .env.example .env

# تحميل نماذج الذكاء الاصطناعي
ollama pull llama3.2
ollama pull nomic-embed-text

# تشغيل التطبيق
python app.py

🎮 الاستخدام

اختصارات لوحة المفاتيح

الاختصار	الوظيفة

Ctrl + Enter	إرسال الرسالة
Ctrl + N	بدء جلسة محادثة جديدة
Ctrl + M	تشغيل/إيقاف مراقب النظام
Ctrl + T	تغيير السمة
Ctrl + /	لوحة الأوامر
Escape	إغلاق النوافذ



---

أمثلة أوامر

/system status
عرض حالة النظام

/model switch llama3.2
تغيير نموذج الذكاء الاصطناعي

/memory clear
مسح ذاكرة المحادثة

/theme quantum-glass
تغيير السمة البصرية

/predict on
تفعيل محرك التنبؤ
